Custom Element Registry

Interactive Demo - Load and Register Web Components

About This Demo

What does this demonstrate?

This demo shows how to use the customElementRegistry function to dynamically load and register custom web components. It demonstrates that you can execute code after all components are loaded, making it perfect for lazy-loading components or building modular applications.

Control Panel

Components to Load:

  • <greeting-card> - A greeting card component
  • <custom-button> - An interactive button with click counter
  • <status-badge> - A status badge component
